Why Kuickboard?

Shape Recognition

Draw rough rectangles or circles with the pen tool and watch them snap into perfect geometry instantly.

Persistent Storage

Never lose your work. Everything is saved locally in your browser using localStorage.

Speed Focused

Designed for power users with keyboard shortcuts for colors, modes, and instant text clearing.

Keyboard Shortcuts

ALTToggle RECTANGLE / PEN mode
CTRL + OChange mode to CIRCLE
CTRL + LChange mode to LINE
CTRL + KToggle colors
CTRL + QClear text inside hovered shape
CTRL + RReset canvas
CTRL + ZUndo last action
